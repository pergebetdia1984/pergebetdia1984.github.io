Puppy Linux can be used for several purposes:

  • As a traditional desktop for getting work done on a desktop tower, laptop or netbook computer
  • As a tool for recovering data, by booting to Puppy Linux and clicking on the icon for the hard drive and then copying documents onto an external device, such as a USB drive
  • As a tool for testing hardware

Getting Puppy Linux

If your computer has a DVD drive you can pick up a Puppy Linux DVD at the NCF office or download the ISO file from PuppyLinux.org and burn it to a DVD yourself with an ISO writer. If your computer does not have an optical drive, you can run it from a USB drive, as described below.

Puppy Linux actually does not need to be 'installed' like a traditional operating system, instead it is normally run it from a DVD or a USB device. It all loads into your computer's RAM for use and the settings and documents all saved at the end of the session. You can run it on a Windows computer, without changing the Windows installation on the hard drive. In fact Puppy can be run on a computer that doesn't even have a hard drive installed.

Using Puppy Linux

This explains how to use Puppy Linux from a Windows PC:

Run Puppy Linux (if you have a DVD already)

  1. Complete a back-up of your bookmarks
  2. Boot to Puppy Linux (many computers require entering the computer set-up on boot to allow it to boot to a DVD before the hard drive)
  3. You can access your existing documents from the hard drive, by clicking on the icon on the bottom of the desktop.
  4. Install bookmarks in Firefox Ctrl-Shift-O → Import and back up
  5. Add desired applications from the 'Install' icon on the desktop or by downloading .pet files directly from the Puppy repositories.
  6. Once you have it running you can create a USB to run it from instead of a DVD, by clicking on 'Install' on the desktop.

Run Puppy Linux from your own DVD

  1. Download the Puppy Linux ISO file from the Puppy repositories
  2. Download the ISO recorder at http://isorecorder.alexfeinman.com/#/
  3. Verify the integrity of the downloaded ISO file by comparing its SHA256 sum to the one for your release on http://distro.ibiblio.org/puppylinux/. This ensures that the ISO has not been compromised or corrupted.
  4. Launch the ISO recorder and use it to burn the ISO image to your DVD
  5. Complete a back-up of your bookmarks
  6. Boot to Puppy Linux (many computers require entering the computer set-up on boot to allow it to boot to a DVD before the hard drive)
  7. You can access your existing documents from the hard drive, by clicking on the icon on the bottom of the desktop.
  8. Install bookmarks in Firefox Ctrl-Shift-O → Import and back up
  9. Add desired applications from the 'Install' icon on the desktop or by downloading .pet files directly from the Puppy repositories.
  10. Once you have it running you can create a USB to run it from instead of a DVD, by clicking on 'Install' on the desktop.

Puppy Linux installation (with your own USB)

  1. Download the Puppy Linux ISO file from https://Puppy Linux.me/downloads/
  2. Download the UNetBootIn ISO USB recorder at http://unetbootin.github.io/
  3. Verify the integrity of the downloaded ISO file by comparing its SHA256 sum to the one for your release on http://distro.ibiblio.org/puppylinux/. This ensures that the ISO has not been compromised or corrupted.
  4. Launch UNetBootIn ISO USB recorder and use it to unpack the ISO image onto your USB
  5. Complete a back-up of your bookmarks
  6. Boot to Puppy Linux (many computers require entering the computer set-up on boot to allow it to boot to a DVD before the hard drive)
  7. You can access your existing documents from the hard drive, by clicking on the icon on the bottom of the desktop.
  8. Install bookmarks in Firefox Ctrl-Shift-O → Import and back up
  9. Add desired applications from the 'Install' icon on the desktop or by downloading .pet files directly from the Puppy repositories.
