Subject:

After the installation of DSS V7 or JovianDSS, it requires activation. How can I perform it?

Contents:

In order to use the DSS V7 or JovianDSS product for more than 30 days you will need to activate it. There are two methods to perform this:

Online
NOTE: Using “Online” method requires to provide Internet access for the Open-E DSS V6/V7 or JovianDSS.

In order to activate the product online access the GUI located in 'HELP > About Data Storage Software V7', check the 'Online activation' and click 'apply'. Application will automatically connect to our server and perform the activation process.


In order to activate the JovianDSS go to 'About and select online activation'. Application will automatically connect to our server and perform the activation process.

In case of any activation failures, please verify if You meet there requirements :

- open following ports in firewall : 25 and/or 10444
- check if default gateway is set and if it's correctly (use ping function under console CTRL+ALT+T to check if activation servers: respond)
IP Address of the activation server :
66.185.18.141 or 46.4.97.13
Offline

NOTE: In this case Internet access for the Open-E DSS V7 is not required.

In order to activate the DSS V7 offline access the GUI located in 'HELP > About Data Storage Software V7', check the 'Offline activation' and click 'download'. Hardware stamp file will be downloaded on you computer. Next step is to send this file to activation@activation-key.com as an attachement (replay with activation file should come with in 7 days).


In order to activate JovianDSS in offline mode, access the GUI. Click on About, select offline activation and download the Hardware stamp. The next step is to send the file to activation@activation-key.com as an attachement. After this our support team will respond with an activation file that You can upload in the same section where You downloaded the Hardware stamp previously.

Important note! The Hardware Stamp file is not assigned to the PCI extension cards and RAM memory. It is possible to replace them without the need for re-activation.
