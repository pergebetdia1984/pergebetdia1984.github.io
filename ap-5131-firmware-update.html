FirmwareVersionChangelog
Latest FW*RUT2XX_R_00.01.13.12020.11.19
Factory FW**RUT2XX_R_00.01.13.12020.11.19

*Latest FW - firmware candidate for release. This version will usually contain some new features and/or improvements (described in the Change log section of this article) but it hasn't been fully tested and approved by Teltonika's testing and technical support units. If Latest is the same version as Factory - it just means that no newer firmware than a Factory version is publicly available.

**Factory FW - firmware used in mass production.
