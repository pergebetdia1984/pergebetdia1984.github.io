Have been playing about with getting it set up so that it manages both my unRAID server and iMac. Found APCUPSD which works on both, after doing some research on how to set up though it would post my config to help anyone looking at something similar.

In order for this config to work you must have an ethernet port or SmartCard installed and you also need the router plugged into the UPS so that devices can still communicate during power outage.

unRAID Config (Master, plugged into UPS via USB cable, ethernet cable also configured)

  • UPSCABLE: USB
  • UPSTYPE: USB
  • Device: (empty)
  • Battery level: 40
  • Minutes: 5
  • Timeout: 0

iMac Config (Slave, uses ethernet)

Unraid Delete Share Mac

  • UPSCABLE: ether
  • UPSTYPE: net
  • Device: 192.168.1.80:3551
  • Battery level: 50
  • Minutes: 10
  • Timeout: 0
  • UPSCLASS: standalone
  • UPSMODE: disable
  • POLLTIME: 0

Device IP is the IP of my unRAID server not the UPS IP. The port used is the default port unless otherwise changed.
